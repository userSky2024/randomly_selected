API文档

端点 方法 描述 返回的JSON数据
/configs GET 获取所有配置 { "1": {"title": "配置1", "options": ["选项1", "选项2"]}, "2": {"title": "配置2", "options": ["选项A", "选项B"]} }
/configs/<config_id> GET 根据ID获取特定配置 { "title": "配置1", "options": ["选项1", "选项2"] }
/configs POST 添加新配置 请求示例 { "title": "标题", "options": [ "选项1", "选项2", ... ] }
返回 { "message": "配置添加成功" }
/configs/<config_id> DELETE 删除特定配置 { "message": "配置删除成功" }
/random/<config_id> GET 从特定配置中随机选择一个选项 { "config_id": 1, "random_option": "选项1" }
